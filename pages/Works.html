ჩემი ნამუშევრები

Hello World

ეს არის ჩემი პირველი HTML ვებგვერდი, სადაც გამოვიტანე წარწერა „Hello World“ და რომლის საშუალებითაც მოვხვდი ამ კურსებზე.

Happy Birthday

„Hello World“-ის გაკეთების შემდეგ, ჩემი სურვილით შევქმენი „Happy Birthday“-ის მისალოცი ბარათი.

მოგზაურობა საქართველოში

აღმოაჩინე ულამაზესი ბუნება საქართველოში. ეს არის ვებპროექტი მოგზაურობაზე საქართველოში, სადაც აღწერილია ლამაზი ადგილები და გამოყენებულია HTML და CSS დიზაინი.

Discover Greece

ეს არის ვებპროექტი სახელად „Discover Greece“, რომლის დიზაინიც შემუშავებულია Figma-ს გამოყენებით. HTML და CSS კოდით ავაწყე დიზაინის ზუსტი ვერსია, რომელიც სრულად ერგება სხვადასხვა ეკრანის ზომას.
